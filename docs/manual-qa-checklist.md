# 手動 QA チェックリスト

コードレビューでは検出できない、目視・操作による品質確認項目です。
リリース前と大型データ更新後に実施してください。

---

## 1. ナビゲーション

| 確認項目 | 期待動作 |
|----------|----------|
| ヘッダーのロゴをクリック | トップページへ遷移 |
| 全ナビリンクを順にクリック | 対応ページへ遷移、アクティブ状態が変わる |
| ブラウザの「戻る」ボタン | 前のページに戻る |
| 存在しない URL を入力 | 404ページが表示される |

## 10. アクセシビリティ（最低限）

- [ ] Tab キーでフォーカスが全インタラクティブ要素を通過する
- [ ] スコアバーに `role="progressbar"` + `aria-valuenow` がある
- [ ] 画像・SVGに `aria-hidden` または `alt` テキストがある
- [ ] 色だけで情報を伝えていない（スコアにラベルも付与）
