# 公開リポジトリ ポリシー

## 基本方針

このリポジトリは **公開前提** で運用します。
以下のポリシーを全貢献者・運営メンバーが遵守してください。

---

## 含めて良いもの

- `data/processed/` — 整形済みの公開統計データ（出典明記済み）
- `data/manual/scores.json` — 編集部が算出したカスコア（方法論は `docs/data-sources.md` に記載）
- `data/raw/` に保存する生データ — **e-Stat / RESAS の公開 API から取得したものに限る**
- ソースコード・設定ファイル
- `.env.example` — 環境変数のテンプレート（値なし）

## 含めてはいけないもの

- **API キー・トークン・パスワード** — `.env` ファイルや直書きを禁止
- 未公表・内部資料 — 市役所や関係団体から非公開で入手した資料
- 個人情報 — 氏名・住所・電話番号等が特定できるデータ
- スクレイピングで取得した著作物の全文
- 未確認の推測データ（「〜と思われる」程度の根拠）

---

## データ掲載基準

| 基準 | 内容 |
|------|------|
| 出典 | 公的機関・オープンデータのみ（`docs/data-sources.md` に記載） |
| 確認 | 数値は原典と突合して確認済みであること |
| 鮮度 | 最新年度のデータを優先（古いデータは `updated` フィールドで明示） |
| 推計 | 推計値は `isEstimate: true` フラグで区別 |

---

## .gitignore で除外するもの

```
.env
.env.local
.env.*.local
data/raw/private/
*.secret
```

プッシュ前に `git diff --cached` で機密情報が含まれていないことを確認してください。
